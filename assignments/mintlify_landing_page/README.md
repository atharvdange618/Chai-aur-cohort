# Mintlify Landing Page Assignment

**Hosted Link:** [https://mintlify.atharvdangedev.in](https://mintlify.atharvdangedev.in)

This project is a recreation of the Mintlify landing page, focusing on replicating the structure, styling, and responsiveness of the original.

## Sections Recreated

The following major sections of the landing page have been recreated:

- **Header:** Navigation bar with the Mintlify logo, main navigation links (Resources, Documentation, Customers, Blog, Pricing), and action buttons (Contact sales, Start for free).
- **Hero Section:** The prominent introductory section featuring an announcement pill, main title, subtitle, email signup form, and a visual UI preview.
- **Logo Garden:** A section displaying logos of various companies.
- **Features Section:** Highlights key features of the platform, structured in a bento grid layout, covering:
  - "Built for both people and AI"
  - "Self-updating knowledge management" (Agent)
  - "Intelligent assistance for your users" (Assistant)
- **Enterprise Section:** Focuses on enterprise-grade features, including a header with title/subtitle and CTA, detailed enterprise features, a customer story card (Anthropic), and a row of enterprise client logos.
- **Customers Section:** Showcases customer success stories from different industries with visual cards and a carousel navigation.
- **Call to Action (CTA) Section:** A concluding section with a prominent title, subtitle, CTA buttons, and resource blocks for pricing and quickstart.
- **Footer:** Contains the footer header with logo and social links, a grid of footer navigation links (Explore, Resources, Documentation, Company, Legal), an enterprise security badge, and a bottom bar with status indicator, copyright information, and theme toggles.

## Fonts Used

- `Segoe UI`, `Tahoma`, `Geneva`, `Verdana`, `sans-serif`: These are the primary font families used, with `Segoe UI` likely being the preferred choice if available, followed by other sans-serif fallbacks.
